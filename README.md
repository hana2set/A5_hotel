## 기능명세

1. 고객 로그인 (입력되어있는 데이터)
2. 메인 화면 (예약, 예약확인, 취소) 
   1. 예약
      1. 호실 입력
      2. 예약 확인
         * 예약 가능 여부 확인 (금액, 예약된 객실 등 확인)
         * 예약 성공 시 UUID 및 예약 정보 반환
   2. 예약확인
      * 고객 예약정보 전체 출력
      * 마스터 계정일 경우 전체 출력
   3. 취소
      1. UUID 입력
         * 해당 정보 출력후 취소 확인
      2. 뒤로가기
